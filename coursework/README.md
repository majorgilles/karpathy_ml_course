# Follow-along coursework

This directory is the working area for completing the lesson issues. Each issue maps to one lesson folder with the same internal layout so it is easy to start, review, and compare progress.

## Standard lesson layout

```text
coursework/<lesson>/
├── README.md      # issue links, goal, workflow, deliverables
├── notebooks/     # exploratory notebooks while following the video
├── src/           # cleaned lesson code and reusable modules
├── tests/         # lightweight checks and gradient/shape tests
└── artifacts/     # small plots, samples, generated text, or metrics
```

Each lesson has a starter notebook with exactly one intro Markdown cell.
